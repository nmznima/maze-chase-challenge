# Maze Chase

A Pac-Man-like game and an online 2D level editor for designing its levels.
See `technical-challenge-frontend-engineer.md` for the task.

## Quick start

```bash
bun install && uv sync
uv run backend
bun run game
```

to run the editor:
```bash
bun run editor
```
